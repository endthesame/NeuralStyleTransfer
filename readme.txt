Чтобы запустить проект нужно иметь установленный:
1)- PIL (консольная команда для установки: pip install Pillow)
  - tensorflow (консольная команда для установки: pip install tensorflow)
  - Matplotlib (консольная команда для установки: pip install matplotlib)
  - numpy (консольная команда для установки: pip install numpy)
  - jupyter notebook (консольная команда для установки: pip install notebook)

2) Открыть Neural Style Transfer.ipynb в jupyter notebook
3) Запустить каждый блок

В папке с проектом предоставлены фотографии для использования: 
-все изображения с "source" используются как content image
-все изображения с "style" используются как style image
Важно: желательно использовать изображения content и style с одним разрешением, так как в папке предоставлены изображение с разным разрешением


Параметры, которые можно менять:
- contentImage 
- styleImage

- content_weight - параметр a - насколько важен нам контент на формируемом изображении
- style_weight - параметр b - насколько важен нам стиль на формируемом изображении

- num_iterations #кол-во итераций
- image.save('results\result7.jpg') - внутри кавычек указываем путь и название, чтобы сохранить результат
